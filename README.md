# .dotfiles

My personal dotfiles for macOS

## Setup

- Clone the repo
- Run the setup script

```bash
git clone
cd .dotfiles
./setup.sh
```

## What's included?

- [Brewfile](https://brew.sh/)
- [Starship.toml](https://starship.rs/)
- Zshrc
- Iterm2 profile
- [Zed settings](https://zed.dev/)
- Setup script
